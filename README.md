# 🪙 Bitcoin: La Cadena del Tiempo

Infografía interactiva sobre la historia completa de Bitcoin — desde la criptografía de 1976 hasta la Reserva Estratégica de EE. UU. en 2025-2026.

## ✨ Características

- **Cronología vertical interactiva** con más de 20 hitos históricos
- **4 idiomas**: Español, English, 日本語, 中文 (cambio instantáneo, se recuerda la preferencia)
- **Panel de datos en vivo**: precio BTC, comisiones, bloque actual y mempool (vía [mempool.space](https://mempool.space))
- **Moneda 3D interactiva**: arrastrable, con inercia y doble-clic para giro rápido
- **Alertas de nuevo bloque** en tiempo real
- **Sección de datos curiosos** sobre el supply de Bitcoin
- **Tarjetas expandibles** con animación al hacer scroll
- Un solo archivo HTML autocontenido — sin build, sin dependencias

## 🚀 Ver en vivo

Abre `index.html` directamente en tu navegador, o despliega en cualquier hosting estático (GitHub Pages, Cloudflare Pages, Netlify, Vercel).
